# Template — Spam / irrelevant

**Action:** `close` — no reply. Log only.

- Marketing pitches, SEO/agency spam, phishing, bot messages, irrelevant content.
- Do **not** reply. Mark the channel item resolved/closed and log to ClickUp as `spam` for volume tracking.
- If a message looks like spam but contains a real order number or payment/legal language, **do not** treat as spam — reclassify (likely `complaint_escalation` or a genuine query).
